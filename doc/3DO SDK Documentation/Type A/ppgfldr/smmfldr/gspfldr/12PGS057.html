ta_envelope

Tests various envelope options by passing test index.

Synopsis

ta_envelope [<test code>]

Description

Demonstrates creating, attaching, and modifying two envelopes which are attached to sawtooth instruments.

  • Button A starts and releases voice A.
  • Button B starts and releases voice B.
  • Button C toggles the states of voices A and B.
  • Control Up and Control Down change the time scaling of the envelope.

Arguments

test code
Integer from 1 to 13, indicating the number of the test. See the source code for what each test actually does. Defaults to 1.
