Peripherals Overview

Peripherals refers to peripheral game-play devices that plug into the control port of a 3DO Interactive Multiplayer system and into the 3DO Development Station. Some peripherals can be daisy-chained, that is, hooked up in series for games that use multiple peripherals.

Development of a peripherals interface for any application must be undertaken solely at the developer's risk and expense. The 3DO Company expressly disclaims any liability or responsibility with respect to such matters.

Overview

This document presents an overview of the kinds of peripherals available for the 3DO Interactive Multiplayer system, and offers some guidelines for developers incorporating peripherals support in titles development.
