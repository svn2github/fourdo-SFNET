FPointFromFVal

Creates an FPoint from inline values.

Synopsis

FPoint * FPointFromFVal (FPoint *dst, frac16 x, frac16 y)

Description

Stores the specified x and y values into *dst. It is useful for writing inline calls like the following: 

FPoint    wrk; 
OffsetCelListByFDelta (cel, FPointFromFVal (&wrk, my_x, my_y));

Arguments

dst
Pointer to the FPoint where the results are to be stored.
x y
Values to be used for the FPoint.

Return Value

The input parameter dst.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, celutils.h
