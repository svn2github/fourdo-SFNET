TimerServicesShutdown

Shuts down the TimerServices thread.

Synopsis

void TimerServicesShutdown (void)

Description

Shuts down the TimerServices thread. Before calling this function, ensure that your task and all its child threads have called TimerCancel() to release any timers they were using.

Only the task which started the TimerServices thread may shut it down.

Return Value

Zero on success or a negative error code.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, timerutils.h
