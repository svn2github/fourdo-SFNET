CenterIPointInDisplay

Returns the center point of the display.

Synopsis

IPoint* CenterIPointInDisplay(void)

Description

Returns the center point of the display.

The return value points to a static structure which must not be modified.

Return Value

Pointer to an IPoint describing the center point of the display.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, celutils.h
