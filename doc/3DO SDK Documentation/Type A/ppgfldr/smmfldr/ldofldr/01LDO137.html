OffsetCel

Adds offsets to a CCB's position fields.

Synopsis

void OffsetCel (CCB *ccb, int32 xOffset, int32 yOffset)

Description

Adds the specified offsets to the position fields of the CCB identified by ccb.

Arguments

ccb
Pointer to the CCB of the cel.
xOffset
Integer x offset.
yOffset
Integer y offset.

Associated Files

lib3do.lib, celutils.h
