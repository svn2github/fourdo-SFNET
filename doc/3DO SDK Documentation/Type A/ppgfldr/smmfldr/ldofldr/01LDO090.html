GetTextCelPenNumber

Retrieves text cel pen number.

Synopsis

void GetTextCelPenNumber (TextCel *tCel, int32 *penNumber)

Description

Retrieves pen number used for text rendering.

Arguments

tCel
Pointer to a TextCel structure.
penNumber
Pointer to a variable to receive the pen number.

Associated Files

lib3do.lib, textlib.h
