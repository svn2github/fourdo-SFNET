CenterSRectInSRect

Centers an SRect within an SRect.

Synopsis

SRect * CenterSRectInSRect (SRect *dst, SRect *src)

Description

Centers the destination rectangle (dst) over or within the source rectangle (src). If the destination rectangle is smaller than the source rectangle, its position values are modified to center it within the source rectangle. If the destination rectangle is larger than the source rectangle, its position values are modified to center it over the source rectangle. In either case, the resulting rectangle retains its original sizes; only the position changes.

Arguments

dst
Pointer to the SRect to be centered.
src
Pointer to the SRect within which dst is to be centered.

Return Value

Returns dst.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, celutils.h
