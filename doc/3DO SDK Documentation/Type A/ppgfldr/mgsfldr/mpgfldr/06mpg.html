Advanced Sound Player

This chapter describes the advanced sound player, which is a new Music library component for playing sound files. The advanced sound player is intended to replace the sound file player discussed in Playing Sound Files . This chapter contains the following topics:

  • Overview of the Advanced Sound Player
  • Players, Sounds, and Markers
  • Decision Functions
  • Caveats
  • For More Information
  • Function Calls
The discussions in this chapter assume that you are familiar with the sound player described in Playing Sound Files .
