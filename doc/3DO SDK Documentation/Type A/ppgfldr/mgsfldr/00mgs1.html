Programming 3DO Music and Audio

This book is a compilation of two manuals that describe the Portfolio Audio and Music folios. The following paragraphs briefly describe each manual.

3DO Music and Audio Programmer's Guide describes the Music and Audio folios. It includes overviews, programming tutorials, sample code, and function call definitions.

3DO Music and Audio Programmer's Reference contains a detailed description of the calls that make up Portfolio Music and Audio folios.
