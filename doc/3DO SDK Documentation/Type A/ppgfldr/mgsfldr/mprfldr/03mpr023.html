fixedstereo16swap.dsp

Play a stereo sample with byte swap.

Function

This instrument plays a stereophonic sample at 44,100 hertz. The audio CD player uses it to swap the bytes as they come off the Red Book CD.

Knobs

Amplitude - 0 => $7FFF

Inputs

None

Outputs

RightOutput
