dcsqxdmono.dsp

Play a square/xact/delta/monocompressed sample.

Function

This instrument plays a square/xact/delta/monocompressed sample at a fixed rate. This is a 2:1 compression. The compression is done by SquashSnd or SoundHack.

Knobs

Amplitude - 0 => $7FFF

Inputs

None

Outputs

Output FIFOs
