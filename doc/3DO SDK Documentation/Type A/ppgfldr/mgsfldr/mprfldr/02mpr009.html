DestroyObject

Destroys an object.

Synopsis

int32 DestroyObject( COBObject *Object )

Description

This function is part of the Juggler object-oriented toolbox that supports the various classes of Juggler objects. This function gets rid of an object. Execute the object's Term() method to close the object, then use this call to get rid of the object by deleting its COBObject data structure and freeing the memory used to store it.

Arguments

Object
A pointer to the COBObject data structure for the object.

Return Value

This function returns 0 if successful or an error code (a negative value) if an error occurs.

Implementation

Library call implemented in music.lib V20.

Associated Files

cobj.h, music.lib
