filterednoise.dsp

Generate filtered noise.

Function

This instrument passes white noise through the state variable filter. It is useful for wind sounds, or other whooshing sounds.

Knobs

Frequency- currently nonlinear, 0=> $4000, default $2000

Resonance - 1 / Q, 0 => $6000, default $1000

Amplitude- 0 => $7FFF
