square.dsp

Square waveform generator.

Function

This instrument is a simple square wave generator. It has a woody, saxophone like sound.

Knobs

Frequency - 0 => $7FFF, default = $300 = raw frequency. If you call TweakRawKnob() the units are phase increments. If you call TweakKnob() the units are 16.16 fractional Hertz.

Amplitude - 0 => $7FFF

Inputs

None

Outputs

Output

Output FIFOs
