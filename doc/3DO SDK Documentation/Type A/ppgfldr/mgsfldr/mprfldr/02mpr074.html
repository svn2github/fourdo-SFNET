spGetMarkerPosition

Get position of an SPMarker.

Synopsis

uint32 spGetMarkerPosition (const SPMarker *marker)

Description

Returns frame position of the SPMarker.

Arguments

marker
Pointer to SPMarker to interrogate.

Return Value

Frame position of the SPMarker.

Implementation
