DeleteCue

Deletes an audio cue created by CreateCue().

Synopsis

Err DeleteCue (Item Cue)

Description

This macro deletes an audio cue and frees its resources (including its associated signal).

Arguments

Cue
The item number of the cue to delete.

Return Value

The function returns zero if successful or an error code (a negative value) if an error occurs.

Implementation

Macro implemented in audio.h V21.

Associated Files

audio.h
