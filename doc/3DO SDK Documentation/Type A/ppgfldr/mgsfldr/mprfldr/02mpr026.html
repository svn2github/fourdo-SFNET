DeleteScoreContext

Deletes a score context.

Synopsis

Err DeleteScoreContext( ScoreContext *scon )

Description

This function deletes the ScoreContext data structure along with all of its attendant data structures, including the NoteTracker data structures used for dynamic voice allocation.

Arguments

scon
Pointer to the ScoreContext data structure to be deleted.

Return Value

This function returns 0 if successful or an error code (a negative value) if an error occurs.

Implementation

Library call implemented in music.lib V20.

Associated Files

score.h, music.lib

See Also

InitScoreMixer(), CreateScoreContext()
