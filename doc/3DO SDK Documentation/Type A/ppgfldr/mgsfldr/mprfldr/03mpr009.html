dcsqxdstereo.dsp

Play a square/xact/delta/stereocompressed sample.

Function

This instrument plays a square/xact/delta/stereocompressed sample at a fixed rate. This is a 2:1 compression. The compression is done by SquashSnd or SoundHack.

Knobs

Amplitude - 0 => $7FFF
