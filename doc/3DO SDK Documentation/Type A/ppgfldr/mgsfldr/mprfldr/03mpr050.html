sawenv.dsp

Sawtooth generator with amplitude envelope.

Function

This instrument is a simple sawtooth generator with a built-in envelope. It is a test instrument that we use to test the envelope code, but you may find it useful as a synthesizer voice.

Knobs

Frequency - If you call TweakRawKnob(), the units are phase increments, If you call TweakKnob(), the units are 16.16 fractional hertz.

Amplitude - 0 => $7FFF
