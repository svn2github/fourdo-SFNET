3DO Music and Audio Programmer's Reference

3DO Music and Audio Programmer's Reference describes the graphics calls and macros the Graphics folio.

This book includes the following chapters:

  • Audio Folio Calls, provides a list of the Audio folio function calls.
  • Music Library Calls, provides a list of the Music library function calls.
  • Instrument Templates, provides a list of the DSP instrument templates available to the Audio folio.
