impulse.dsp

Impulse waveform generator.

Function

This instrument is a simple impulse generator. The output is either zero or the Amplitude value. The width of the impulse is one sample. This is useful for pinging filters.

Knobs

Frequency - 0 => $7FFF, default = $300 = raw frequency. If you call TweakRawKnob() the units are phase increments, If you call TweakKnob() the units are 16.16 fractional Hertz.

Amplitude - 0 => $7FFF

None
