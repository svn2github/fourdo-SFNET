envfollower.dsp

Track the contour of a signal.

Function

This instrument tracks the positive peaks of an input signal. It decays exponentially based on the droop factor. It outputs a fairly smooth signal that can be used to control other signals.

Knobs

Droop - -$0 => $7FFF, default = $7FF0 = number to multiply previous result by to generate exponential decay. Higher numbers give slower decay.

Input

Outputs

Output
