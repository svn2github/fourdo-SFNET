Installing 3DO PostPro

If your Macintosh is running System 7.5, you can skip this section. System 7.5 installs these files automatically during its installation.

If your Macintosh is running System 7.1 or earlier, follow these steps:

  1. Make sure AppleScript 1.1 is in your Extensions folder.
  2. Make sure the 3DO PostPro program and its Remote folder are in the same folder (see Figure 1).

    Figure 1: 3DO PostPro program and Remote folder icons

  3. Place the Macintosh Drag and Drop file and the Dragging Enabler file in your System folder (see Figure 2).

    Warning: A Clipping extension file is included in the same directory. If you are running System 7.1 or later, delete the Clipping extension file, as it causes system problems.

    Figure 2: Dragging Enabler and Macintosh Drag and Drop icons

  4. Restart your system.
When you have successfully installed 3DO PostPro, a blank Workbench appears.
