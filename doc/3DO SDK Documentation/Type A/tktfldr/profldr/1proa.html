About 3DO PostPro

3DO PostPro helps developers prototype and produce content for 3DO titles. Its primary function is to help in the second stage of the content production process; converting art to 3DO format. To help facilitate this process, PostPro provides a number of basic functions:

  • Opening and saving various file types
  • Converting files
  • Editing files with data-specific editors
  • Previewing files on the 3DO system and a television monitor, with interactive editing display
