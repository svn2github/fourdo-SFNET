Converting Files

This document shows how to convert non-3DO files to 3DO formats. You can convert Photoshop RGB to 3DO coded cel types, and Photoshop indexed files to 3DO coded cel types.

This document covers how to convert files one at a time, or many files at once using a batch conversion process. AppleScript is used as the batch processing utility; example scripts are provided.
