Working with 3DO Images

A 3DO image is the most basic 3DO graphic. This chapter steps you through creating 3DO images for both NTSC and PAL screens.

How to use this document

If you're just interested in creating an image, skip introductions and go directly to the "How to" section of a section. This chapter isn't meant to be read front to back.

Note: Some of the information is meant for programmers only. If you are a graphic artist, focus on the process and skip some of the explanations.
