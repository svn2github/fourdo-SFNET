The 3DO Options menu

This section discusses the 3DO Options menu, which is not available in Studio/32. Figure 1 shows all commands in the 3DO Options menu:

Figure 1: 3DO Options menu.

Access 3DO Palette command

When you select Access 3DO Palette, that command is checked, the 3DO palette becomes the default palette, and the other commands in the 3DO Options menu become available.

The command is a toggle: If you select it again, the other commands are dimmed and the Studio/32 palette becomes the default palette.

Show/Hide 3DO Palette command

When you select Show/Hide 3DO Palette (or type 3), 3DO Animator displays (or hides) the Set 3DO Object Type dialog that lets you create 3DO images and cels. This dialog is discussed in detail in The Set 3DO Object Type Dialog.

Figure 2: Set 3DO Object Type dialog.

Set CCB command

When you select Set CCB, 3DO Animator displays the Load CCB Settings dialog that lets you do the following:

  • Load CCB settings from existing files.
  • Go to other dialogs that let you:
  • Customize CCB settings.
  • Add special effects with translucency options.
  • Turn packing on and off.
  • Choose a format for saving the file.

Figure 6: Load CCB Settings dialog.

Translucency Options command

When you select Translucency Options, 3DO Animator displays the Translucency Options dialog.

Figure 7: Translucency Options dialog.

The dialog lets you do the following:

  • Set the P-mode.
  • Adjust the opacity.
  • Enhance luminance.
  • Add shadows.
  • Access the 3DO Cel Parameters dialog by using the More Options dialog.
These special effects are achieved by mixing the cel pixels with the image pixels. The selections you make for the first source and second source influence the outcome of the mixing. Likewise, the multiply and divide choices you make will also influence the effects you achieve.

For tutorial-style examples that illustrate the effects of these options, see Creating 3DO Images and Cels.
