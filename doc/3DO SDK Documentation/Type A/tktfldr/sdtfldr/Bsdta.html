File names

All samples in this collection conform to the following name convention:

Name.Description.aiff

For example: Piano.A5LM44k.aiff

The description part of the filename can include the following:

--------------------------------------------------------
Name Element         |Discussion                        
--------------------------------------------------------
Note Name (if        |Based on middle C = C4 (not the   
applicable)          |Yamaha C3)s = sharp (no flats     
                     |used)                             
--------------------------------------------------------
Loop Type(s)         |L = sustain loopLR = release and  
                     |sustain loop                      
--------------------------------------------------------
Mono/Stereo          |M = monoS = stereo                
--------------------------------------------------------
Sample Rate          |44k (44,100 kHz)22k (22,050 kHz)  
                     |(not Mac 22k)Vk (variable rate)   
--------------------------------------------------------
