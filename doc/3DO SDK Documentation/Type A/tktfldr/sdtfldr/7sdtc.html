Adjusting instrument knobs

As you listen to a patch, use the control pad to switch pages, adjust knobs, and start and release the patch. The control pad controllers are given in the following table

Table 1:  Using the control pad to adjust instrument knobs.
--------------------------------------------------------
Select         |To...                                   
--------------------------------------------------------
Up             |Move up through faders on the current   
               |page.                                   
--------------------------------------------------------
Down           |Move down through faders on the current 
               |page.                                   
--------------------------------------------------------
Left           |Decrease the selected fader's value in  
               |large increments.                       
--------------------------------------------------------
Right          |Increase the selected fader's value in  
               |large increments.                       
--------------------------------------------------------
Left Shift +   |Decrease the selected fader's value in  
Left           |small increments.                       
--------------------------------------------------------
Left Shift +   |Increase the selected fader's value in  
Right          |small increments.                       
--------------------------------------------------------
A              |Toggle between starting and releasing a 
               |patch.                                  
--------------------------------------------------------
Left Shift + A |Stop patch                              
--------------------------------------------------------
B              |Start a patch when the button is        
               |pressed, release a patch when the button
               |is released (similar to a key on a MIDI 
               |keyboard).                              
--------------------------------------------------------
C              |Cycle forward through instrument pages. 
--------------------------------------------------------
Left Shift + C |Cycle backwards through instrument      
               |pages.                                  
--------------------------------------------------------
Right Shift    |Puts you into oscilloscope mode (see    
               |Table 2).                               
--------------------------------------------------------
X (Stop)       |Quit PatchDemo.                         
--------------------------------------------------------

Note: Fader values are raw values (a value the knob can actually use), so frequency values are not in Hz. See the 3DO Audio Programmer's Guide for more information on raw values.

Using Oscilloscope mode

If you select Right-Shift, you are put in oscilloscope mode and different key bindings apply. 

Table 2:  Control pad input in oscilloscope mode. 
--------------------------------------------------------
Input                |Result                            
--------------------------------------------------------
Move cross left/right|Changes X scale of wave display   
--------------------------------------------------------
Move cross up/down   |Changes Y scale of wave display   
--------------------------------------------------------
Shift left/right     |Scrolls wave                      
--------------------------------------------------------
A button             |Refresh                           
--------------------------------------------------------
C button             |Audio Monitor                     
--------------------------------------------------------
