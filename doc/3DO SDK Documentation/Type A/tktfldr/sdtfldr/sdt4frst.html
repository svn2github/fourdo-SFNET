Creating MIDI Projects with ARIA

This document explains how to use ARIA to play standard MIDI files already saved on disk, using a 3DO PIMap (program-instrument map) file. You also learn how to use ARIA in conjunction with a MIDI manager-compatible sequencer to create music in real time.
