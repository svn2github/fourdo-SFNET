-----------------------------------------------------

-----------------------------------------------------

-----------------------------------------------------

-----------------------------------------------------
rednoise             |Red Noise                      
-----------------------------------------------------
sawtooth             |Sawtooth Wave                  
-----------------------------------------------------

-----------------------------------------------------

-----------------------------------------------------
fixedmonosample      |16-bit fixed-frq mono

timesplusunsigned    |a x b + (unsigned) c
