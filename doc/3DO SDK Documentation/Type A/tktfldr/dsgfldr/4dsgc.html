Performing fast-forward and fast-rewind operations

A new sample application, ShuttlePlayer, performs fast-forward and fast rewind operations using an adapted version of DSGoMarker(). For more information, see the source code in the Streaming:Examples folder.

Note: The ShuttlePlayer sample application actually represents a different branch in streaming: It has several options to DSGoMarker() that are not found in the base DataStreamer library. See the 3DO Release Notes for more information.
