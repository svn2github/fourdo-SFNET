3DO Debugger Quick Start

The 3DO Debugger allows you to run, trace, and step through a program on the 3DO system. The Debugger provides a window to the 3DO world: You can execute any program residing on the Macintosh, and you can use the extensive debugging facilities, performance analyzer, CD-Emulation facility, and other features to successfully complete the development process. This document provides quick access to all important Debugger functionality. Click on any topic below to learn about it:

  • Starting a debugging session
  • Using display windows to examine the source code
  • Moving through the program efficiently
