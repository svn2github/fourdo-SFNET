setpri

Sets the shell's priority.

Synopsis

setpri <priority>

Description

Sets the priority of the shell's task. Sometimes it is desirable to boost the shell's priority to a high number. This lets commands such as showtask or memmap work with more accuracy. Use the showshellvars command to display the current priority of the shell.

Arguments

priority
The new shell priority; this value must be in the range 10..199.

Implementation

Command implemented in shell V20.

Built-in shell command.

Synonyms

shellpri

See Also

showshellvars
