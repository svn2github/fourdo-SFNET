showerror

Displays an error string associated with a numerical system error code.

Synopsis

showerr <error number>

Description

Displays the error string associated with a numerical system error code.

Arguments

error number
The numerical error code for which you want to see the string. This number can be in decimal, or in hexadecimal starting with 0x or $.

Implementation

Command implemented in shell V21.

Built-in shell command.

Synonyms

err
