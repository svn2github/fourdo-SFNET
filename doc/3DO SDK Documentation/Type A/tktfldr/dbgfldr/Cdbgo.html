showavailmem

Displays information about the amount of memory currently available in the system.

Synopsis

showavailmem

Description

Displays information about the amount of memory installed in the system, and the amount of memory currently free.

This command also displays the amount of memory currently consumed by supervisor-mode code.

Implementation

Command implemented in shell V20.

Built-in shell command.

Synonyms

availmem

See Also

showmemmap, showmemlist
