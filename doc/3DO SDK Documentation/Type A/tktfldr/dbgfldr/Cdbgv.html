showtask

Displays information about a task in the system.

Synopsis

showtask [task name | task item number]

Description

Displays information about task and threads in the system. If you supply a specific task or thread name, a more detailed output is produced describing the specified task exclusively.

Arguments

<task name | task item num>
Name or item number of a task. The item number can be in decimal or hexadecimal format starting with 0x or $. If this argument is not supplied, then general information about all tasks in the system is displayed.

Implementation

Command implemented in shell V20.

Built-in shell command.

Synonyms

ps
