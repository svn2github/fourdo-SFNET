DumpStreamCCBs

Prints formatted summary of CCBs in a SANM stream.

Synopsis

dumpstreamccbs [options] -i <filename>

Description

DumpStreamCCBs is an MPW tool to dump cel control blocks (CCB's) found in an SANM stream.

Arguments

-i <filename>
Input stream file name.
-s <time>
Start time for dump.
-xf
Hex file position output.
-fbs
Filler block sizes output only.
-cf
Dump nonstream cel file.
-APOS
Dump APOS chunks (default). Must be typed all uppercase.
-ACCB
Dump ACCB chunks (default). Must be typed all uppercase.
-AMAP
Dump AMAP chunks. Must be typed all uppercase.

Examples

Here is a typical command line:

dumpstreamccbs -i newSANM
