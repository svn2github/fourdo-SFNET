InitDataStreaming

Initializes data streaming.

Synopsis

int32 InitDataStreaming (long maxNumberOfStreams)

Description

Initializes the DataStreamer environment by allocating the memory pool required for the context blocks of the specified number of data streams. InitDataStreaming() allocates a memory pool large enough to contain a context block for each stream, whether or not the maximum number of streams is initialized.

Arguments

maxNumberOfStreams
Maximum number of streams that may be created.

Return Value

Value
Error Condition
kDSNoErr
None
kDSNoMemErr
Could not allocate memory pool

Associated Files

DataStream.h
