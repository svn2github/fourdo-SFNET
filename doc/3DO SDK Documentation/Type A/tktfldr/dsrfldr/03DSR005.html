DumpAIFF

Prints formatted summary of AIFF/AIFF-C chunk data.

Synopsis

dumpaiff -i <filename>

Description

Prints a formatted summary report of AIFF and AIFF-C chunk data.

Based on the chunk type, the following information is included:

  • chunk type
  • chunk size
  • number of channels
  • number of sample frames
  • sample size
  • sample rate
  • number of markers
  • base frequency
  • detune
  • low-frequency
  • high-frequency
  • low-velocity
  • high-velocity
  • gain

Arguments

-i <filename>
Input file name.

Examples

Here is a typical command line:

dumpaiff -i bach_44K_16B_M.aiff
