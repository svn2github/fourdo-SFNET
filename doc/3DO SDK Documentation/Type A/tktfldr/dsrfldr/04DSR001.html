audioclockchan

Specifies channel for designated clock source.

Synopsis

audioclockchan <audio_clock_channel>

Description

Lets you specify which audio channel is designated as clock source.This information is included in the stream header of the file if writestreamheader is also included in the script.

Arguments

audio_clock_channel
A mask that lets you specify the channel that uses the clock. Give this mask in hex, for example, to specify channel 2, specify 04. There is a total of 31 maskable channels.

Examples

audioclockchan 04

Caveats

None.
