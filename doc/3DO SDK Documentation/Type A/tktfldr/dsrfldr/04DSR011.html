streamdeltapri

Specifies the streamer thread delta priority.

Synopsis

streamdeltapri <delta_priority>

Description

Specifies the streamer thread delta priority.

The delta_priority should be low because the task priority of the streamer thread is calculated in relation to the priority of its parent task

Arguments

delta_priority
Integer. If you add the priority of the application that uses the data acquisition thread and the delta priority, the number must not exceed 200. The higher the number, the higher the priority.
