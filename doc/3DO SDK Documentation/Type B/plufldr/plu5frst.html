3DO Custom Code Module for DeBabelizer

To convert more than one piece of source art at a time, you need a tool with batch processing capabilities like DeBabelizer. This document discusses the available Custom Code Modules (CCMs) for DeBabelizer to create animations or streamed animations using the 3DO CCM or the Coded-8 CCM.
