GetMsgPortSignal

Returns the signal mask for a message port.

Synopsis

uint32 GetMsgPortSignal (Item msgPortItem)

Description

Returns the signal mask for a message port.

Arguments

msgPortItem
Item of the message port whose signal mask is returned.

Return Value

Value
Error Condition
(The signal mask for the message port)
0
Could not locate message port

Associated Files

MsgUtils.h
