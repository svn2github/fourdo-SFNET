subscriber

Specifies which subscriber will be instantiated.

Synopsis

subscriber <subscriber_ID> <delta_priority>

Description

Lets you add the subscriber entry in the stream header and specify which subscriber will be instantiated. Each subscriber entry consists of the subscriber ID (subscriber data type) for the subscriber and a delta priority to be used when the subscriber is instantiated. If you use this command, you need to also include the writestreamheader command.

delta_priority should be low because the task priority of the subscriber thread is calculated in relation to the priority of its parent task.

Arguments

subscriber_ID
4-byte ASCII characters describing the subscriber type (FILM, SNDS, CTRL, SANM).
delta_priority
0 to 200. The higher the number, the higher the priority.
