writestopchunk

Writes a STOP control chunk.

Synopsis

writestopchunk <stream_time>

Description

Writestopchunk writes a STOP control chunk to the output stream. The stream time specifies the time where the STOP control chunk is to be placed in this stream.

This command produces the same output as CtrlMaker.

Arguments

stream_time
0 to 7FFFFFFF (hex)

Caveats

None.

See Also

markertime, writegotochunk, writehaltchunk
