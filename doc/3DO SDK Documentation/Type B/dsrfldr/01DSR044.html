InitDataAcq

Initializes the data acquisition memory pool.

Synopsis

int32 InitDataAcq (int32 dataAcqCount)

Description

Initializes the memory pool used by acquisition threads. InitDataAcq() must be called before any acquisition threads are created with NewDataAcq().

Arguments

dataAcqCount
Number of acquisition threads for which the context memory pool is initialized.

Return Value

Value
Error Condition
kDSNoErr
None
kDSNoMemErr
Could not allocate memory pool for acquisition threads

Associated Files

DataAcq.h

See Also

InitDataStreaming, CloseDataAcq, NewDataAcq
