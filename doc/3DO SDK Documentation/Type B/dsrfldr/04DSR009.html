streamblocksize

Specifies physical blocking factor for this stream.

Synopsis

streamblocksize <block_size>

Description

Specifies the physical blocking factor for this stream. The block size must be a multiple of 2048 bytes. Block sizes less than 32,768 bytes are not recommended if a 300 KB/second data rate is desired.

Arguments

blocksize
0 to 7FFFFFFF (hex) (multiple of 2048 bytes)
