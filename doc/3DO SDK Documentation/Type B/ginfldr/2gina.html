Finding the right tool

This section lists each 3DO graphics type or effect and the tools to create it.

To find out which graphics type is appropriate for a given situation, see Preparing Art for 3DO Titles.

3DO image (16-bit)

3DO Image Writer plug-in, 3DO PAL Image Writer plug-in, 3DO PostPro, 3DO Animator, ppmto3DO (MPW tool)

3DO image (24-bit)

ppmto3DO (MPW tool)

3DO uncoded cel

3DO CelWriter plug-in, 3DO Coded-8 CelWriter plug-in, 3DO CCM for DeBabelizer, 3DO Animator, 3DO PostPro, makecel (MPW tool)

3DO coded cel

3DO CelWriter (palette problems possible), 3DO CelWriter CCM for DeBabelizer, 3DO Coded-8 CelWriter plug-in, 3DO Animator, 3DO PostPro, makecel (MPW tool)

Transparent background (no black in cel)

3DO CelWriter plug-in, 3DO Coded-8 CelWriter plug-in, 3DO Animator, 3DO PostPro

Transparent background (packed cel, black allowed)

3DO Coded-8 CelWriter plug-in, 3DO Animator, 3DO PostPro

Anti-aliased cel

3DO Anti-Alias CelWriter

Fonts

3DO FontWriter

Animations

3DO Animator, 3DO CCM for DeBabelizer

Video

MovieCompress, 3-2 Pull-Down, MovieEdit
